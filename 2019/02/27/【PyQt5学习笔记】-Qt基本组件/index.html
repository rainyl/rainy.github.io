前言

最近由于有课程需要做GUI界面，但是不想用课程所讲的C#与Winform，老师也允许使用其它语言与框架，因此使用了PyQt5与Python3来写。

担心自己隔一段时间会忘记在这个过程中遇到的坑，因此想到写这么一个系列的学习笔记来记录下来。

主要是记录遇到的坑，关于基础的内容也不需要太多记录，随便搜一下就有了

Qt组件

layout

layout是指布局方式。主要继承自PyQt5.QtWidgets。

坑1

刚开始，由于没有做过什么界面开发，因此对于代码到底是怎么变成我们看见的界面的并不清楚，我就是不明白我们写了那么几句话，怎么就变成了界面，虽然看着各位大佬的文章，也有图片，但是始终不如自己去写一遍。也许就是因为不清楚，才会对布局感到迷惑，这也是第一个坑。

恩？好像不对，第一个坑应该是不知道各种组件到底叫什么。我当时是先找了一些介绍基础的书看了一下，并且跟着例子敲，运行，先对整体有了概念，至少知道了什么叫什么，才开始做自己的程序。
